## 特殊机制

### 第一阶段

每把剑单独出现，当血量小于66%时切换到下一把剑（每把剑一次）

顺序：诛仙剑->戮仙剑->陷仙剑->绝仙剑

### 第二阶段

每两把剑一起出现，当血量小于33%时切换到下一把剑（每把剑一次）

顺序：诛仙剑+戮仙剑->陷仙剑->绝仙剑

### 第三阶段

四把剑一起出现

## 噩梦

需要通过四剑之后再打四剑归一

| 四剑归一技能                                                 |
| ------------------------------------------------------------ |
| 连斩：连续斩击附近的敌人,伤害的50%转化为自身血量             |
| 横扫四方：水平蓄力，之后以极快速度横扫                       |
| 屠戮四方：转动剑并追踪目标方向移动，对碰到的目标附加死亡诅咒Buff |
| 诛仙剑阵：剑虚影分成4剑,生成剑阵，对剑阵中的敌人造成伤害（沉默+减速） |
| 剑魂护身：剑魂环绕周身，此时攻击会反弹50%伤害，并给剑回血    |
| 万剑归宗：剑飞空中，召唤漫天飞剑                             |
