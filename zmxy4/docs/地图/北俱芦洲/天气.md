## 冻天

| 天气电之祖巫技能                                             |
| ------------------------------------------------------------ |
| 利爪连舞：挥舞前爪连续攻击前方的玩家                         |
| 春回雾暖：召唤一团雾气，雾气可以被打散，玩家站在雾气里命中会下降，BOSS站在雾气里会持续回血 |
| 冬雪严寒：耳朵上的青蛇幻化出一个雪蛇图腾，雪蛇会召唤大雪，若10秒内不被破坏，将冻结全场玩家 |
| 夏阳酷暑：另一只耳朵上的青蛇幻化出一个烈蛇图腾，烈蛇发出烈日的光芒，被光芒笼罩的玩家生命回复、法力回复将下降 |
| 秋风为虐：从远端刮过一阵旱风，吸入BOSS口中，被刮到的玩家生命和法力上限将会下降 |

掉落装备：奢天古铃制作书
