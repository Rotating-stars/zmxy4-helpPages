PC端 ​

浏览器+flash ​

官方的网页游玩方式，游戏网址如下

造梦西游4：https://www.4399.com/flash/zmhj.htm?g=4

copy
https://www.4399.com/flash/zmhj.htm?g=4

首先需要下载并安装flash，以下是我个人使用的flash纯净版三合一安装包，提取码：qyqk

flash：https://1825022173.share.123pan.cn/123pan/2rR2jv-5Dc23

然后需要一个支持flash的浏览器，我推荐使用旧版本的世界之窗，提取码：qyqk

世界之窗：https://1825022173.share.123pan.cn/123pan/2rR2jv-pFB23

造梦微端 ​

官方的游戏启动器，下载链接如下

造梦微端：https://dl.4399ytzg2.com/microgame/4399zmxyonline.exe?from=zmxyblockflashtip&1777450939

下载之后安装即可

造梦盒子 ​

第三方开源的游戏启动器，支持二倍速，快捷须弥山面板，跳过斩杀

请点击链接前往Gitee，从releases页面选择最新的版本（版本号最高）

下载页面：https://gitee.com/duskeye/zmBox/releases

然后选择文件后缀为.zip的文件下载，下载之后记得解压

造梦盒子（Mac） ​

基于原有duskeye/造梦盒子项目，转译以供macOS以及IOS使用

请自行参照项目指示，完成对应安装

项目页面：https://gitee.com/b1ame/zmbox-for-macos-ios

360游戏大厅 ​

从以下链接进入官网，然后点击右上角下载客户端并安装

官网：https://360game.360.cn/

安装完成后千万别点别的东西，点击上方中间我的，然后在左下角找到添加游戏

名称填入造梦西游4，网址填入下方链接，然后就可以在左侧看见游戏图标了

copy
https://www.4399.com/flash/zmhj.htm?g=4
